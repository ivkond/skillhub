<objective>
Migrate the app shell and high-reuse UI primitives to semantic token styling with dark-mode parity, while removing legacy decorative effects from core product surfaces.
</objective>

Wave 1 foundation artifacts stay unchanged in intent; Wave 2 consumes them to migrate shell + shared primitives + auth/dashboard baseline.
</wave_boundary>
