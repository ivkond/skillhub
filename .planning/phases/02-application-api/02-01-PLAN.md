<objective>
Deliver **DTOs**, **portal application services**, and **REST controllers** that expose Phase 1 collection domain behavior over HTTP with **VIS-01..VIS-03**-safe responses and COL/ROL/INT-01/INT-02 semantics preserved from domain services.
</objective>

<threat_model>
**OWASP ASVS L1 (collections HTTP)**

| Risk | Mitigation |
|------|------------|
| A01 Broken access control | Every mutating path passes `actingUserId` from `@RequestAttribute("userId")` (authenticated routes); domain `SkillCollectionAuthorizationPolicy` remains authoritative. |
| A03 Injection | Use `@Valid` DTOs, `Long` path variables, no raw SQL in controllers. |
| A04 Insecure design | Public read split into dedicated controller with explicit permitAll policy added in plan **02-02**; this plan only implements handlers. |
| A05 Security misconfiguration | Controllers do not disable CSRF; follow `NamespaceController` session patterns. |
</threat_model>
